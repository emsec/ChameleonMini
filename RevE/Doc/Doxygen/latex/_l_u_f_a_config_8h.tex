L\-U\-F\-A Library Configuration Header File.  




\subsection{Detailed Description}
L\-U\-F\-A Library Configuration Header File. This header file is used to configure L\-U\-F\-A's compile time options, as an alternative to the compile time constants supplied through a makefile.

For information on what each token does, refer to the L\-U\-F\-A manual section \char`\"{}\-Summary of Compile Tokens\char`\"{}.
